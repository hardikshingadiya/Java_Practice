# Java_Practice
A repository of Java programs to improve coding and problem-solving skills.

           ------------------

Requirements to run a Java program:

JDK (Java Development Kit) – Used to write, compile, and run Java programs.

JRE (Java Runtime Environment) – Provides the environment to execute Java programs.

Text Editor or IDE – Used to write Java code (e.g., Notepad, VS Code, Eclipse).

Java Source File – The program should be saved with the .java extension.

Compiler and Interpreter – Use javac to compile the program and java to run it.
For Example : 
  File Name - Example.java
  To Compile the Java File :-
        javac Example.java
  To Run the Java File :-
        java Example


           ------------------

Note : The filename must be the same as the public class name and should have the .java extension.

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

File name must be Hello.java

           ------------------

Java Naming Conventions :

Class names: Use PascalCase (Upper CamelCase).
✅ Student, BankAccount, HelloWorld

❌ student, bank_account

Method names: Use camelCase.
✅ getName(), calculateSum(), displayResult()

❌ GetName(), calculate_sum()

Variable names: Use camelCase.
✅ studentName, totalMarks, age

❌ StudentName, student_name

Constant names: Use UPPER_CASE with underscores.
✅ MAX_SIZE, PI, DEFAULT_VALUE

Package names: Use lowercase.
✅ com.example.app, java.util
